Ruby Cheatsheet

Standard Classes

  • Integers 3
  • Floats 2.95
  • Strings "anything in quotes"
  • Booleans true/false
  • Symbols (literal sequence of characters):symbol
  • Ranges [1..6]
  • Arrays [1,"a",[3,4]]
  • Hashes {key:"value","key2":[1,2,3]}

Operators

Math

  • Addition: a + b
  • String addition: "a"+"b"="ab"
  • Subtraction: a - b
  • Multiplication: a * b
  • String multiplication: "a"*5="aaaaa"
  • Division: a / b
  • Integer division: 1 / 2 = 0
  • Remainder/modulus: a % b

Comparison

  • Equals: ==
  • Less than: <
  • Greater than: >
  • Not equals: !=
  • Less than or equal to: <=
  • Greater than or equal to: >=
  • Included in: ===
  • Less than/equal to/greater than (returns -1/0/1): <=>
  • Boolean And: &&
  • Boolean Or: ||
  • Ternary operator (equivalent to if A then B else C) A ? B : C

Defining a class:

class Something
	@instance_variable = value
	def method...
end

Create a new object of that class:

object = Something.new(args)

Call a method of that class:

object.method(args)

Flow Control

Defining methods:

def method_name(arguments) do
	code block
	return value
end
When not using "return", a code block will return the value of the last expression evaluated.

Calling methods:

method_name(args)
or for object/class-defined methods...
object.method_name(args)
Code blocks can be enclosed with {}, or with "do" and "end".

Conditionals:

if condition {block 1}
elsif {block 2}
else {block 3}
end

Case switches:

case letter
	when "a" {block 1}
	when "b" {block 2}
	else {block 3}
end

Looping

while condition {block}
until condition {block}
for counter in range {block}

Recursion

def method_name() do
	break if condition
	method_name()
end

Iterating on Objects

Integers

integer.times {block}

Arrays

array.each {|value| block}
Strings can be iterated over like arrays!

Hashes

hash.each {|key,value| block}

Ranges

[start..end].each {|value| block}
